Теги

Назва тегу < td >

  Призначений для створення однієї комірки таблиці. Тег < td > повинен розміщуватися всередині контейнера < tr >, який в свою чергу розташовується всередині тега < table >.

Синтаксис

< table >

  < tr >

    < td > ... < /td >

  < /tr >

< table >

Атрибути

  • abbr   Короткий опис вмісту комірки
  • align   Визначає вирівнювання вмісту комірки
  • axis   Групує комірки зв'язані між собою схожою інформацією
  • background   Задає фоновий малюнок в комірці
  • bgcolor   Колір фона в комірці
  • bordercolor   Колір рамки
  • char   Вирівнює вміст комірки по заданому символу
  • charoff   Зміщує вміст комірки відносно заданого символу
  • colspan   Об'єднує горизонтальні комірки
  • headers   Дозволяє зв'язувати комірки з заголовком
  • height   Висота комірки
  • nowrap   Забороняє перенос строк
  • rowspan   Об'єднує вертикальні комірки
  • scope   Дозволяє зв'язати комірки простої таблиці в речових браузера
  • valign   Вирівнювання вмісту комірки по вертикалі
  • width   Ширина комірки
